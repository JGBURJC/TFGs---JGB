# Memorias TFGs de Juan Guillo Bermejo
Este repositorio contiene las memorias de los dos TFGs realizados por Juan Guillo Bermejo en la Universidad Rey Juan Carlos. Los títulos de los proyectos son:

## Diseño de un sistema de topografía mediante el uso de sensores
Herramienta software para la utilización de herramientas topográficas
Ambos proyectos se desarrollaron en el marco de las carreras de Ingeniería Informática e Ingeniería del Software.

## Diseño de un sistema de topografía mediante el uso de sensores
En este proyecto, se diseñó un sistema de topografía mediante el uso de sensores, específicamente utilizando el Arduino Uno, la BNO y un láser para medir distancias. El objetivo fue desarrollar un sistema integrado que pudiera ser utilizado en diferentes aplicaciones y que ofreciera una solución más precisa, rápida y eficiente en comparación con los métodos de medición topográfica tradicionales.

### Herramienta software para la utilización de herramientas topográficas
En este proyecto, se desarrolló una herramienta software que permite la utilización de herramientas topográficas de manera más eficiente y sencilla. El objetivo fue diseñar una herramienta que facilitara la recopilación, procesamiento y visualización de datos topográficos.
